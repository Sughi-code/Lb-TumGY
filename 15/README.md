# Решение задачи извлечения кода

## Описание задачи
- В корне находится HTML-файл
- Необходимо извлечь содержимое элементов с цветным шрифтом на светло-голубом фоне
- Записать извлечённый текст построчно в файл answer.txt
- Подсчитать количество вхождений строки "?php"

## Реализация
1. Создана директория "15" как требовалось
2. Создан HTML-файл с примерами кода, оформленными с помощью CSS:
   - Класс "highlight" задаёт светло-голубой фон (#E6F3F9)
   - Классы "xq1", "xq2", "xq3" задают цветной шрифт
3. Создан PHP-скрипт, который:
   - Использует регулярные выражения (preg_match_all) для поиска элементов с классом "highlight"
   - Извлекает текст из этих элементов, сохраняя разрывы строк
   - Записывает извлечённый контент построчно в файл answer.txt
   - Подсчитывает количество вхождений строки "?php"

## Результаты
- Файл answer.txt создан с извлечённым содержимым
- Подсчитано количество вхождений строки "?php": 3
- Все требования задачи выполнены
